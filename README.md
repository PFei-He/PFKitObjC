安装 Install
--- 
#### 手动安装 Manually
```
将`PFKit`文件夹拖入到工程目录中，导入文件`PFKit.h`
```

#### CocoaPods
```
target 'YourTarget' do
    platform:ios, '7.0'
    pod 'PFKitObjC', '~> 0.0.1'
end
```

使用 Use
---
#### CocoaPods
```
#import <PFKitObjC/PFKit.h>
```
